Romania Endpoints API

Date geografice și corporative pentru România
Gratuit & Open Source
API complet pentru județele, localitățile, codurile poștale și companiile din România.
Include coordonate, date despre populație, informații geografice și verificare CUI-uri prin ANAF.

📊 Despre proiect

Acesta este un proiect conceput pentru orice tip de date, dar în prezent se concentrează pe datele geografice și corporative românești. API-ul include acum și funcționalitate de verificare a companiilor prin CUI (Cod Unic de Înregistrare), integrându-se direct cu baza de date ANAF pentru informații actualizate.

Pentru mai multe funcționalități, aștept tickete pe GitHub. Oricine dorește să extindă proiectul este mai mult decât binevenit să contribuie pe GitHub!
